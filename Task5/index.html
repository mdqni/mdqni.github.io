JavaScript Overview

JavaScript is the programming language of the web. It allows you to create dynamic and interactive web pages.

  • Runs in the browser or on server (Node.js).
  • Manipulates HTML and CSS dynamically.
  • Handles user interactions and events.

Example:

function greet(name) {
    alert("Hello, " + name + "!");
}

greet("Madi");

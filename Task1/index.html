About This Assignment

This assignment demonstrates how to build a basic website layout using HTML and CSS without modern layout tools like Flexbox or Grid. You will learn how to:

  • Use colors, fonts, margins, and padding effectively.
  • Understand and apply the box model for layout.
  • Create navigation bars, sidebars, and content sections using divs.
  • Structure content clearly for better readability and design.

Assignment Instructions:

  1. Create a header with a website title.
  2. Add a navigation bar with links to other tasks.
  3. Build a sidebar with a list of courses or topics.
  4. Fill the main content area with descriptions, instructions, or articles.
  5. Add a footer with your name and copyright information.

After completing this task, you should have a solid understanding of basic layout and styling techniques in CSS and be ready for more advanced web development tasks.
